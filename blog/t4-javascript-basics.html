JavaScript Basics

How does JavaScript compare to HTML and CSS?

JavaScript is smarter and much more alive/active. JavaScript has the capacity to respond to user input and interaction, and has the power to make calculations and modify output dependant on the input. It's a much more dynamic language.

Explain the difference between accessing data from arrays and objects

You access data from arrays using the array index (numbers starting from zero that are associated with each item in the array). Objects have Key:value pairs, meaning object properties can be accessed by using their property name or key.

Additionally, object data can be accessed using dot notation and bracket notation. Arrays rely solely on bracket notation.

Explain what functions are and why they are useful

Functions take a block of code and package into a function that can be called and repeated when required. For the 'having breakfast' example above, if that were not a function I would need to write out these steps every morning. If we created a havingBreakfast function, I would only need to call that function by writing havingBreakfast();.

Functions share the benefits of frameworks in that they remove the need for developers to re-write common code over and over again.

Additionally, functions can take arguments to produce varying output based on the input. This is super neat and makes it really powerful and just downright nifty.
