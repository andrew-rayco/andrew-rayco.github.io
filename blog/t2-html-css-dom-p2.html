HTML, CSS & the DOM II

What is grid based design?

Grid based design uses a combination of rows and columns to layout a page structure. The grid can vary depending on the desired layout, both site-wide and on a row by row basis. Commonly a 12 column layout is used as 12 can be divisible by 1, 2, 3 and 4 so it allows for flexibility.

Whats all the hype about responsive webpages?

Given the wide variety of web-accessible devices nowadays, webpages need to be flexible enough to accommodate screens of all sizes. A device-agnostic, responsive site or app will ensure acceptable usability, whatever the display size.

What is semantic structure?

Semantic structure means applying meaningful markup to content. Elements are marked up according to their importance (e.g. Headings and paragraphs), or type of content (e.g. lists and images).

This is particularly important with regards to accessibility (ensuring content is screen-reader friendly) and with delivering effective search engine results.
