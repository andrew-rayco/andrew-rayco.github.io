JavaScript

In the JavaScript language, describe all the use cases for each of:

parentheses ()

Parentheses can be used in a few different ways in JavaScript. Firstly in a more traditional mathematics sense, whereby a statement or operation can be compartmentalised, and processed only within itself and before any outside computations are acted upon it.

They also execute a function, and can contain arguments for the function, although arguments are not always required. Parentheses are also used when initialising a function, to identify the code as a function, and also to contain any parameters.

brackets []

Brackets can be used to initialise an array, or to access nested properties in an object or an array.

braces {}

Braces can be used in JavaScript to enclose a block of code. Often as part of an if or while statement, or a loop.

Braces are also used to initialise an object.

single quotes '' and double quotes ""

Quotes are used to contain a string. These are used throughout JavaScript, including as arguments in functions, and binding elements by class or id.

There's a very good chance there are exceptions to this, but on the whole there is no difference between single and double quotes in JavaScript. As long as you use the same type to start and finish a string, you can use either.
