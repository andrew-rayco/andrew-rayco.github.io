HTML
The HTML is the pure content, what your website says. It should be presented semantically, that is, where content elements are marked up according to their importance (e.g. Headings and paragraphs), or type of content (e.g. lists and images).
CSS
The CSS is the presentation, how your website looks. Colours, typography, borders etc.
The DOM
The DOM (Document Object Model) is a little trickier to define. It's a tree-like structure (the model) of the current document where each element within is represented by a node (like a person in a family tree), and how they are all related. The DOM can be manipulated by JavaScript.

What is meant by boxifying design?

Essentially boxifying is to break down a design into small boxes, each representing a specific element on the page. All websites are structurally built from boxes that together form a grid.

What is the box model?

The Box Model defines the ultimate size of a box by calculating each of the four elements. From inside out, the elements are:

  • content
  • padding
  • border
  • margin
